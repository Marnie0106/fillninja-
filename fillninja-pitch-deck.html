The Solution
Meet FillNinja.

FillNinja is a Chrome extension + AG2 backend with distinct agents: a Search Curator that finds relevant application URLs from the open web, and Fill agents (planner + safety reviewer per tab) that execute in parallel.

Your pitch shifts from “autofill this form” to “I want 20 scholarships — go find them and complete them.” Discovery, orchestration, and execution are one workflow.

Intelligent DOM Scanning
Reads every form field, label, and context on the page via content script
Web discovery
Curator agent + live search turns your goal into a shortlist of real forms to complete
Parallel fill fleet
Multiple tabs, multiple SSE-backed fill runs — batch scholarships or apps without serial clicking
Privacy-First Design
Your data stays local. Only field metadata sent to the agent — never your values
How It Works
4 steps. Zero effort.
01
Discover
Curator agent searches the web and ranks URLs that likely host real applications
02
Parallelize
Extension opens parallel tabs; each page gets a DOM snapshot for its Fill agent
03
Plan
Per tab: AG2 planner proposes actions; reviewer approves before execution
04
Execute
Content scripts run fills, clicks, and scrolls across all active tabs concurrently
